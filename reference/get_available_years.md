# Get available years of Vermont enrollment data

Returns a vector of school year ends for which Vermont enrollment data
is available through the Vermont Education Dashboard.

## Usage

``` r
get_available_years()
```

## Value

Integer vector of available school years (end years)

## Examples

``` r
get_available_years()
#>  [1] 2004 2005 2006 2007 2008 2009 2010 2011 2012 2013 2014 2015 2016 2017 2018
#> [16] 2019 2020 2021 2022 2023 2024 2025
```
